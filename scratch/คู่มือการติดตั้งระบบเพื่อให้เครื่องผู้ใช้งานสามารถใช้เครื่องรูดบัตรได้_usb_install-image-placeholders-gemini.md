- 1. Software Requirement
- 1.1. Windows XP, Windows 7 (32-bit และ 64-bit) หรือ Windows 10 (32-bit และ 64-bit) (วิธีตรวจสอบเวอร์ชั่นของ Windows ตามข้อ 5)

- 1.3. Driver สำหรับเครื่องรูดบัตร (rxtxSerial.dll)
